This is the backend API for the Inventory Management System, built using Node.js, Express, MongoDB, and Mongoose.
It provides RESTful APIs to manage products, categories, stock, and related inventory operations.


Project Setup & Installation

Follow the steps below to run the backend on your local machine.

1.Clone the Repository
2.Install Dependencies ==> npm install
3.Environment Variables
Create a .env file in the root directory.

4.Start the Server ==> npm start
